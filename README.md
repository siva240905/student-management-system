# 🎓 Student Management System

A modern and responsive Student Management System developed to simplify educational administration. The application enables efficient management of student records, attendance, courses, academic performance, and administrative activities through an intuitive dashboard.

## ✨ Features

- 👨‍🎓 Student Registration & Management
- 📚 Course Management
- ✅ Attendance Tracking
- 📊 Academic Performance Monitoring
- 🔍 Search & Filter Students
- 📈 Dashboard & Statistics
- 📝 Student Profile Management
- 🔐 Secure Login & Authentication
- 📱 Responsive User Interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Vercel (Deployment)

## 🚀 Live Demo

https://student-management-system-chi-five.vercel.app/

## 🎯 Purpose

This project was developed to digitalize and simplify student administration by providing an easy-to-use platform for managing student information, attendance, courses, and academic records. It demonstrates modern web development skills and responsive dashboard design.
